amavisd-new

Homepage: http://www.ijs.si/software/amavisd
Supports: clamd, clamscan
amavisd-new is a rewritten version of amavis maintained by Mark Martinec.
Installation:
clamscan is enabled automatically if clamscan binary is found at amavisd-new starup time. clamd is activated by uncommenting its entry in the @av_scanners list, file /etc/amavisd.conf.
